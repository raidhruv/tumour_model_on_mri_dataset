# Brain Tumor Detection on MRI using CNN 

## Overview

A deep learning-based system for detecting brain tumors from MRI scans using Convolutional Neural Networks (CNN). The model analyzes medical images and predicts the presence of tumors, enabling faster and more accurate preliminary diagnosis.

---

## Features

* CNN-based MRI image classification
* Automated tumor detection
* Image preprocessing and normalization
* Fast inference on medical images
* Model training and evaluation pipeline

---

## Tech Stack

Python • TensorFlow/Keras • OpenCV • NumPy • Matplotlib

---

## Usage

```bash
pip install -r requirements.txt
python train.py
python predict.py
```

---

## How it Works

MRI Image → Preprocessing → CNN Model → Tumor Prediction

---

## Results

* Accurate classification of tumor vs non-tumor MRI scans
* Demonstrates application of deep learning in medical imaging
